Sphinx audEERING themeÂ¶

Sphinx theme to style your documentation project with audEERING colors.

The theme extends the official readthedocs theme by

  • audEERING colors and logo

  • support for wide pages, useful for wide tables

  • better formatting of tables

  • custom styling of figure captions

  • make juptyer-sphinx code cells appear like normal code cells

  • custom styling for copy-button and nbsphinx

Have a look at the installation and usage instructions as a starting point.
